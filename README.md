# Dataset README

## Usage

Road Defect Dataset is now open available. Please download **aliyundrive** to access the dataset. https://www.aliyundrive.com/s/xnoEp4KB5B3

Please note that this dataset is open-source and non-profit. It is not meant for commercial or profit-oriented usage.

## Content

Road Defect Dataset is built by students from Jilin University. The dataset covers pavement contexts in Hebei Province, Henan Province, and Jilin Province, China, including conventional urban roads, highways, and village roads. It covers **four** categories of road surface defects: longitudinal crack, transverse crack, alligator crack and pothole.
